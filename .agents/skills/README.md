# Skills Directory

Use this directory for reusable project-specific skills.

## Skill format

Each skill lives in its own folder:

`skills/<skill-name>/SKILL.md`

Optional supporting files can live beside `SKILL.md`:

- `examples.md`
- `reference.md`
- `scripts/`

## Naming

- Use lowercase kebab-case.
- Keep names specific (`website-seo-audit`, not `helper`).

## Authoring checklist

1. Copy `skills/_template/SKILL.md`.
2. Update frontmatter:
   - `name`
   - `description`
3. Keep instructions short and action-oriented.
4. Include trigger language in the description ("Use when ...").
5. Add examples only when they improve output quality.
